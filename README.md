## 不同点
* 该程序必定会输出当前目录下的东西，也就是说我如果输入`./myls.out ../`程序还是会输出当前目录的文件信息。
* 输出格式不相同，该程序明显不能规范输出，文件名长一点就错位了
* 最重要的是该程序不支持选项，输出的结果是固定的。
